你怎么看这些架构的取舍？在什么场景下你会优先选择 SSR（Next.js/Nuxt）而不是纯 CSR？

请描述一次你独立完成的从 0 到 1 的全栈项目，
重点讲技术选型、关键架构决策、以及上线后暴露出的问题与回顾。

如何在一个全栈项目中统一错误处理？
请从前端 UI 层、Fetch 层、后端业务层、日志层四个维度说明你的实现方式。

请描述你在项目中是如何做鉴权与权限控制的，
包括登录态管理、RBAC/ABAC、接口级权限、前端路由守卫等环节。

全栈开发中如何做端到端的性能优化？
请分别从网络、前端渲染、后端接口、数据库、缓存五层给出具体手段。

你如何设计一套全栈项目的本地开发环境？
涉及到前后端热更新、Mock、环境变量、数据库初始化、种子数据等，请给出你的工程化方案。

请讲一下你理解的 DevOps 和 CI/CD，
在一个前后端同仓（monorepo）的项目里，你会怎么组织构建、测试、部署流水线？

在 Docker 容器化部署场景下，前端静态资源和后端 API 分别怎么打包？
Nginx 反向代理 + CORS 怎么配置？请给出一个典型的 docker-compose.yml 思路。

请描述你做过的最复杂的一次数据库 schema 设计，
涉及的实体有哪些？如何处理一对多/多对多关系？后续是否经历过字段迁移？

如何设计一个"可观测"的全栈应用？
请从日志（logging）、指标（metrics）、链路追踪（tracing）三方面给出方案。

在前后端都使用 TypeScript 的项目中，
你会如何共享类型定义？是走 monorepo + workspace、代码生成、还是 tRPC？

请谈谈你对全栈项目中"安全"的整体认知，

如何实现一个支持文件上传的功能？
要求支持大文件断点续传、秒传、进度条、防止重复上传，请给出整体方案。

请讲一下你对 Serverless（如 Vercel、Cloudflare Workers、阿里云函数计算）的理解，
你会在什么场景下选它而不是传统容器？它的冷启动问题如何缓解？

请描述你在一次真实项目中如何解决跨团队协作的技术冲突，
比如前端期望接口结构 A，后端认为 B 更合理，你最终如何推动决策？

什么是"渐进增强（progressive enhancement）"和"优雅降级"？
请结合你做过的业务场景说明它们的实际价值。

如果让你从零搭一个类似本项目（AI 面试系统）的全栈 MVP，
